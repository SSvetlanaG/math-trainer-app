# 🧮 Математический тренажер

Интерактивное приложение для тренировки навыков арифметики, работы с матрицами и решения квадратных уравнений.

## 🚀 Функционал
- **Арифметика:** Генерация примеров на сложение, вычитание, умножение.
- **Матрицы:** Операции над матрицами (сложение, вычитание, умножение на число).
- **Квадратные уравнения:** Решение квадратных уравнений с выводом корней.
- **Статистика:** Сохранение результатов пользователя по всем разделам.

## 🎮 Меню программы
После запуска вам будет доступно:
1. **Арифметика** — тренировка с числами
2. **Матрицы** — операции над матрицами
3. **Статистика** — просмотр результатов
4. **Квадратные уравнения** — решение квадратных уравнений
0. **Выход** — завершение работы

## 📊 Статистика
Результаты всех тренировок сохраняются автоматически и привязаны к имени пользователя. Для просмотра статистики выберите соответствующий пункт меню.
