# math-trainer-app
# 🧮 Математический тренажер

Интерактивное приложение для тренировки навыков арифметики и работы с матрицами.

## 🚀 Функционал
- **Арифметика:** Генерация примеров на сложение, вычитание, умножение и деление.
- **Матрицы:** Операции над матрицами (сложение, умножение на число).
- **Статистика:** Сохранение результатов пользователя.
